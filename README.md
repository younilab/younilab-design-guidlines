# Youni Lab 設計系統與開發規範

歡迎來到 Youni Lab 設計系統與開發指引文件。本文件庫旨在協助設計師與工程師建立一致的視覺語言與組件標準。

## 📖 文件導覽
* **[設計規範說明 (Design)](DESIGN.md)**: 品牌設計 Token、色彩系統、字體排版、間距、圓角與佈局規範。
* **[組件規範說明 (Components)](COMPONENTS.md)**: 收錄 38 個 UI 元件的詳細規格、狀態變體與行為規則。
* **[開發指引說明 (Guidelines)](Guidelines.md)**: 專案協作流程、設計原則與 AI 提示詞（Prompt）工作流。

* **互動頁面範本網頁**: [younilab-頁面範本.html](younilab-頁面範本.html) (或英文版 `younilab-templates.html`)
